# Geo Pinas

An interactive Philippine map with clickable province borders. Built for fun, free and open source.

## Live Demo

Host the folder on any static server (e.g. GitHub Pages) — no build step required.

## Features

- All 83 Philippine provinces (including Metro Manila as NCR)
- Click a province to see its name and region in the sidebar
- Zoom and pan with mouse/touch or the +/⊙/− buttons
- Collapsible sidebar
- Navy blue chevron-wave ocean background
- Flat/minimal design · No server required

## Tech Stack

- [D3.js v7](https://d3js.org/) — zoom/pan behavior
- Vanilla JS + SVG — no framework
